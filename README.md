🎬 Oldflix

Projeto de catálogo de filmes desenvolvido com o objetivo de praticar desenvolvimento web utilizando HTML, CSS e JavaScript.

A aplicação simula uma plataforma de streaming, permitindo visualizar informações de filmes através de uma interface inspirada em serviços reais de entretenimento.

🚀 Funcionalidades
Exibição de catálogo de filmes;
Organização visual dos conteúdos;
Interface responsiva;
Interação através de JavaScript;
Estrutura de página semelhante a plataformas de streaming.
🛠️ Tecnologias utilizadas
HTML5
CSS3
JavaScript
📚 Conceitos praticados

Durante o desenvolvimento foram trabalhados:

Estruturação de páginas web;
Estilização com CSS;
Manipulação do DOM;
Organização de componentes visuais;
Lógica de programação com JavaScript.

Clone o repositório:
git clone https://github.com/gabrielDuarteAlves7/Oldflix2.0.git
Abra o arquivo:
index.html
